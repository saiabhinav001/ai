# AI Algorithms Collection

This repository contains implementations of various Artificial Intelligence algorithms and techniques in Python.

## Contents

### Search Algorithms
- **BFS.py** - Breadth-First Search implementation
- **DFS.py** - Depth-First Search implementation
- **DepthLimitedSearch.py** - Depth-Limited Search algorithm
- **UniformCostSearch.py** - Uniform Cost Search implementation

### Problem Solving
- **NPuzzleSolver.py** - N-Puzzle problem solver
- **WaterJugProblem.py** - Water Jug problem solver
- **WaterJugProblemUsingAstar.py** - Water Jug problem using A* algorithm

### Constraint Satisfaction Problems (CSP)
- **NQueens.py** - N-Queens problem solver
- **GraphColoring(CSP).py** - Graph Coloring using CSP

### Game Theory
- **Minimax.py** - Minimax algorithm implementation
- **AlphaBetaPruning.py** - Alpha-Beta Pruning optimization

### Probabilistic Models
- **BayesianNetworks(BurglaryAlarm).py** - Bayesian Networks example
- **HMM.py** - Hidden Markov Model implementation

### Notebooks
- **AI_Lab_External.ipynb** - AI Lab exercises
- **Exploring Libraries.ipynb** - Exploration of AI libraries

## Usage

Each Python file can be run independently:

```bash
python BFS.py
python DFS.py
# etc.
```

## Requirements

- Python 3.x
- Additional dependencies may be required for notebooks (Jupyter)
